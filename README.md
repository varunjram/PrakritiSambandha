### Built with:

- ReactJS
- React router
- Prime React UI
- CSS
- Axios
- Mockbee(backend)

### How to install and run locally ?

```
$ git clone https://github.com/varunjram/PrakritiSambandha.git
$ cd PrakritiSambandha
$ npm install
$ npm start
```

## Features -

- User Feed
  - List all user posts
  - Sort by Date & Trending
- Create a Post
- Like a post
- Edit & Delete Post
- User Profile - Add Avatar - Add Bio - Add Portfolio URL
  -Follow/Un-follow
  -Explore Feed
  -Bookmark Post
  -Authentication
  -Sign-up
  -Login
  -Logout
